# クアルト ゲームルール

## 概要

クアルトは2人対戦のボードゲームです。
4×4マスのボードと16個のコマを使い、**ラインを揃えた方が勝ち**というシンプルなルールですが、
「相手にコマを渡す」という独特なメカニクスで高い戦略性を生みます。

---

## コマの属性

各コマは4つの二値属性を持ちます。属性の組み合わせは2⁴ = **16通り**で全て異なります。

| 属性 | 値0 | 値1 |
|------|-----|-----|
| 高さ | 低い (short) | 高い (tall) |
| 色 | 濃い (dark) | 薄い (light) |
| 形 | 四角 (square) | 丸 (round) |
| 頭 | 中実 (solid) | 中空 (hollow) |

### コマのビット表現

コマIDは0〜15の整数で表現します。

```
bit3 (8) : 頭   - 0=中実, 1=中空
bit2 (4) : 形   - 0=四角, 1=丸
bit1 (2) : 色   - 0=濃い, 1=薄い
bit0 (1) : 高さ - 0=低い, 1=高い

例:
  コマ0  (0b0000) = 低い・濃い・四角・中実
  コマ5  (0b0101) = 高い・濃い・丸・中実
  コマ15 (0b1111) = 高い・薄い・丸・中空
```

## ゲームの流れ

```
ゲーム開始
    ↓
Player 1 が「相手に渡すコマ」を選ぶ  ← GIVEフェーズ
    ↓
Player 2 がそのコマを「ボードに置く」 ← PLACEフェーズ
    ↓
  勝利判定 → 勝利なら終了
    ↓
Player 2 が「相手に渡すコマ」を選ぶ  ← GIVEフェーズ
    ↓
Player 1 がそのコマを「ボードに置く」 ← PLACEフェーズ
    ↓
  勝利判定 → 繰り返し
```

**重要**: 「どこに置くか」は自分で決めるが、「何を置くか（コマの種類）」は相手が決める。

---

## 勝利条件

縦・横・斜め いずれかの**4マスが全て埋まり**、かつ**1つ以上の属性を4コマ全てが共有**していれば勝利。

### 判定アルゴリズム

```python
# 4つのコマ a, b, c, d (各0〜15) について:
common_ones  = a & b & c & d        # 全コマが「1」の属性ビット
common_zeros = (~a & ~b & ~c & ~d) & 0xF  # 全コマが「0」の属性ビット
is_win = bool(common_ones or common_zeros)
```

### チェックするライン (計10本)

| 種類 | 数 |
|------|-----|
| 横ライン | 4本 |
| 縦ライン | 4本 |
| 斜めライン | 2本 |
| **合計** | **10本** |

---

## 引き分け

全16マスが埋まっても誰も勝利していない場合、引き分けです。
（非常にまれなケース）

---

## 戦略のポイント

- **相手に危険なコマを渡さない** — 相手が次ターンに勝てるコマを渡してはいけない
- **渡すコマを選ぶ時** — 残り全コマが危険な場合、どれかを渡すしかない（必敗の局面）
- **置く場所を選ぶ時** — 受け取ったコマで勝てるマスを見つけたら即置く
